Social Media Performance Dashboard for Facebook and Instagram Analytics

A Power BI dashboard built for social media performance tracking: followers, posts, reach, engagement, reels views, top content, platform comparison, and date-based trend analysis from API-driven data.

The Problem

Facebook and Instagram data usually stays inside separate platform screens. That makes it difficult to compare reach, engagement, followers, post volume, and top-performing content across both channels without manual exports.

  • Facebook and Instagram performance data needed one reporting layer
  • Manual platform checks made trend tracking slow and inconsistent
  • Stakeholders needed overview, platform-level, content-level, and comparison pages
  • The dashboard needed refresh-ready data instead of static screenshots

The Solution

I built a reporting pipeline that pulls social media data from Meta APIs, stores structured records in SQL, and exposes the analytics layer through Power BI. Fabric Gateway supports refresh-ready reporting for the published dashboard.

  • API extraction for Facebook and Instagram profile, post, reach, engagement, and media metrics
  • SQL tables for platform, daily, and content-level analysis
  • Power BI model with overview, Facebook, Instagram, content, and compare pages
  • Fabric Gateway connection for refresh flow between the dataset and data source
  • Interactive date filters, top content ranking, trend charts, post type breakdown, and KPI cards

Dashboard Pages

  • Overview: total followers, posts, reach, engagement, platform contribution, and top content
  • Facebook: Facebook reach, engagement, post type, daily performance, and post-wise analysis
  • Instagram: followers, reach, engagement, reels views, post count, and media-type performance
  • Compare: side-by-side platform analysis for business decisions

Architecture

Data Flow:
Meta Graph API -> Python/FastAPI service -> SQL database -> Power BI model -> Fabric Gateway -> Published dashboard

Metrics: followers, reach, engagement, likes, comments, shares, reels views, post count
Platforms: Facebook + Instagram
Reporting layer: Power BI dashboard with interactive filters and ranking views

Tech Stack

  • API: Meta Graph API for Facebook and Instagram data
  • Backend: Python, FastAPI, requests/aiohttp, APScheduler
  • Database: SQL database for structured social media metrics
  • BI: Power BI dashboard, DAX measures, interactive visuals
  • Refresh: Fabric Gateway for published reporting workflow

Outcome

One dashboard for Facebook and Instagram performance

The dashboard gives a single view of social media performance across reach, engagement, followers, top content, and platform comparison. It is built for ongoing refresh instead of one-time reporting.
